Get Stuff Done.

Notorious is an app that organizes your tasks and notes.

Download

All in One Place

With Notorious, your tasks and notes are just a tab away. By being so close, they can easily work together.

Advanced Features

Notorious provides half-completions, task attachments, markdown notes and more. Use these features to organize in the best way possible.

Calendar Widget

Other task/note apps typically do not include a calendar widget. Notorious not only includes one, it also includes customization options.

Open Source
